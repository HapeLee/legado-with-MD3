# 仿真二翻页动画实现 - 产品需求文档

## 概述
- **Summary**: 在阅读界面中添加一个新的"仿真二"翻页动画，并在长按翻页区域时显示一个可调整动画速度的弹窗
- **Purpose**: 为用户提供更多翻页动画选择，并允许用户根据个人偏好调整动画速度
- **Target Users**: 喜欢自定义阅读体验的用户

## 目标
- 实现基于SimulationPageAnim.java的仿真二翻页动画
- 在阅读界面中添加新的动画类型选项
- 实现长按翻页区域显示速度调整弹窗
- 弹窗样式符合现有项目UI设计
- 保存用户调整的动画速度设置

## 非目标（范围外）
- 修改现有翻页动画的行为
- 添加其他类型的翻页动画
- 改变现有UI的整体风格

## 背景与上下文
- 项目已有多种翻页动画实现，包括覆盖、滑动、仿真、滚动和淡入淡出
- 用户反馈希望有更多翻页动画选择
- 部分用户希望能够调整动画速度以获得更流畅的阅读体验

## 功能需求
- **FR-1**: 添加新的"仿真二"翻页动画类型
- **FR-2**: 实现基于贝塞尔曲线的翻页效果，参考SimulationPageAnim.java
- **FR-3**: 添加长按翻页区域触发速度调整弹窗的功能
- **FR-4**: 实现速度调整滑块，允许用户在100-500ms范围内调整动画速度
- **FR-5**: 保存用户调整的动画速度设置

## 非功能需求
- **NFR-1**: 动画性能流畅，不影响阅读体验
- **NFR-2**: 弹窗样式与现有项目UI保持一致
- **NFR-3**: 速度调整设置持久化存储
- **NFR-4**: 代码实现符合项目现有代码风格

## 约束
- **Technical**: Android平台，Kotlin语言，使用现有项目框架
- **Dependencies**: 依赖现有项目的动画系统和UI组件

## 假设
- 用户已经熟悉现有的翻页动画选择界面
- 项目已经有完善的设置存储机制
- 现有的弹窗组件可以用于实现速度调整功能

## 验收标准

### AC-1: 仿真二翻页动画类型添加
- **Given**: 用户进入阅读设置
- **When**: 用户选择翻页动画类型
- **Then**: 列表中应包含"仿真二"选项
- **Verification**: `human-judgment`

### AC-2: 仿真二翻页动画效果
- **Given**: 用户选择了"仿真二"翻页动画
- **When**: 用户在阅读界面进行翻页操作
- **Then**: 页面应显示基于贝塞尔曲线的翻页效果，包括阴影和3D效果
- **Verification**: `human-judgment`

### AC-3: 长按速度调整弹窗
- **Given**: 用户在使用"仿真二"翻页动画
- **When**: 用户长按翻页区域
- **Then**: 应显示一个包含速度调整滑块的弹窗
- **Verification**: `human-judgment`

### AC-4: 速度调整功能
- **Given**: 速度调整弹窗已显示
- **When**: 用户拖动滑块调整速度
- **Then**: 动画速度应实时变化，且设置应被保存
- **Verification**: `human-judgment`

### AC-5: 弹窗样式一致性
- **Given**: 速度调整弹窗已显示
- **When**: 用户观察弹窗样式
- **Then**: 弹窗应与现有项目的其他弹窗样式保持一致
- **Verification**: `human-judgment`
