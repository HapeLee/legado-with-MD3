# 仿真二翻页动画完善 - 产品需求文档

## 概述
- **Summary**: 完善仿真二翻页动画功能，使其在阅读设置界面中正常显示，并添加翻页动画速度调节窗口
- **Purpose**: 解决仿真二翻页动画在阅读设置界面不显示的问题，同时为用户提供翻页动画速度调节功能，提升用户阅读体验
- **Target Users**: 所有使用 Legado with MD3 应用进行阅读的用户

## 目标
- 修复仿真二翻页动画在阅读设置界面不显示的问题
- 实现翻页动画速度调节弹窗功能
- 允许用户通过长按翻页区域触发速度调节
- 保存用户的动画速度设置
- 保持与现有 UI 风格一致

## 非目标（范围外）
- 不修改其他翻页动画的行为
- 不改变动画的核心算法
- 不添加新的翻页动画类型

## 背景与上下文
- 项目中已经实现了 `simulationPageAnimV2` 翻页动画（在 PageAnim.kt 中定义为 6）
- 翻页动画的加载逻辑在 ReadView.kt 的 upPageAnim() 方法中已经实现
- 但在阅读设置界面（ReadStyleDialog.kt 和 dialog_read_book_style.xml）中缺少对应的选项
- 用户无法在设置界面选择仿真二翻页动画，也无法调节其速度
- AppConfig.kt 中已经定义了 simulationPageAnimV2Speed 配置项

## 功能需求
- **FR-1**: 在阅读设置界面添加仿真二翻页动画选项
- **FR-2**: 实现翻页动画速度调节弹窗
- **FR-3**: 实现长按翻页区域触发速度调节弹窗
- **FR-4**: 保存用户调节的动画速度设置
- **FR-5**: 在动画速度调节中应用用户设置

## 非功能需求
- **NFR-1**: 弹窗样式与现有项目 UI 保持一致
- **NFR-2**: 速度调节设置持久化存储
- **NFR-3**: 代码实现符合项目现有代码风格
- **NFR-4**: 动画性能流畅，不影响阅读体验

## 约束
- **Technical**: Android 平台，Kotlin 语言，使用现有项目框架
- **Dependencies**: 依赖现有项目的动画系统和 UI 组件

## 假设
- 用户已经熟悉现有的翻页动画选择界面
- 项目已经有完善的设置存储机制
- 现有的弹窗组件可以用于实现速度调节功能

## 验收标准

### AC-1: 仿真二翻页动画在设置界面显示
- **Given**: 用户打开阅读设置界面
- **When**: 用户查看翻页动画选项
- **Then**: 列表中应包含"仿真二"选项
- **Verification**: `human-judgment`

### AC-2: 仿真二翻页动画可选择
- **Given**: 用户在阅读设置界面
- **When**: 用户选择"仿真二"翻页动画
- **Then**: 应用应正确切换到仿真二翻页动画
- **Verification**: `human-judgment`

### AC-3: 长按触发速度调节弹窗
- **Given**: 用户在使用翻页动画阅读
- **When**: 用户长按翻页区域
- **Then**: 应显示翻页动画速度调节弹窗
- **Verification**: `human-judgment`

### AC-4: 速度调节功能正常
- **Given**: 速度调节弹窗已显示
- **When**: 用户拖动滑块调整速度
- **Then**: 动画速度应实时变化，且设置应被保存
- **Verification**: `human-judgment`

### AC-5: 弹窗样式一致性
- **Given**: 速度调节弹窗已显示
- **When**: 用户观察弹窗样式
- **Then**: 弹窗应与现有项目的其他弹窗样式保持一致
- **Verification**: `human-judgment`

### AC-6: 速度设置持久化
- **Given**: 用户已调整动画速度
- **When**: 用户关闭应用后重新打开
- **Then**: 之前设置的动画速度应保持不变
- **Verification**: `programmatic`
